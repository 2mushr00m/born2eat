## 📌 시작 가이드

### 1) 설치

- Node: `v22.21.0`
- npm: `10.9.4`

```text
nvm install 22.21.0
nvm use 22.21.0
```

### 2) 의존성 설치

```text
# (프론트 준비 중) npm run front:install
npm run server:install
```

### 3) 환경변수 설정

```text
cp server/.env.example server/.env
```

### 4) 실행

```text
npm run server:dev      # 서버 시작
```

### 5) 접속 확인

서버 기본 주소: `http://localhost:PORT`
